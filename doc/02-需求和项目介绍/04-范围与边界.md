# 范围与边界

## 1. 本期功能范围（已实现/可用）

### 1.1 业务数据域

- 资产盘点（列表、填报、编辑、删除）
- GYS 供应链/供应商/企业档案
- GAS 供应链/供应商/供应商档案/整车厂
- 供应商画像与产业图谱

### 1.2 任务与采集域

- 自动化采集任务管理
- 任务执行、停止、重试
- 任务日志与质量报告
- 任务结果导入与导出下载

### 1.3 知识与 AI 域

- 知识库文档管理（文件/Web/Text）
- 向量检索页面
- LLM Wiki 工作台/图谱/导入与配置
- LangChain 多功能对话、RAG 对话
- 精准寻源智能体（含 stream）

### 1.4 平台配置域

- MCP 服务管理
- Skill 管理
- 模型提供商管理
- 搜索设置（含配额/测试）
- 数据源连接管理（schema/表预览/SQL）
- 菜单可见性与 UI 样式配置

## 2. 非本期范围

- 细粒度组织权限模型（RBAC）
- 多租户强隔离的独立资源调度
- 分布式任务中心与多节点调度编排
- 全链路可观测平台（Tracing/Metric/Alert 完整体系）

## 3. 边界与约束

- 鉴权依赖外部 Auth 服务，回调地址和端口受限。
- AI 结果受模型能力与外部服务可用性影响。
- 部分智能体页面为占位（准入排查/实时监控/供应商尽调）。
- 文档仅描述代码已实现与明确可验证能力，不对未来能力做承诺。

# 用户与场景

## 1. 用户角色

### 1.1 业务运营

关注点：
- 供应商基础信息与画像完整性
- 供应链关系可追溯
- 产业图谱可读性

常用页面：
- `GYS/GAS` 供应商管理系列
- 供应商画像工作台
- 分析看板

### 1.2 数据工程/数据分析

关注点：
- 数据采集任务执行质量
- 数据导入结果与失败原因
- 数据源连通性与 SQL 验证

常用页面：
- 自动化采集管理
- 数据源管理
- 知识库管理 / LLM Wiki

### 1.3 AI 应用开发

关注点：
- 模型提供商配置
- 工具链（MCP/Skill）可用性
- RAG 与智能体回答质量

常用页面：
- 模型管理、MCP 服务、Skill 管理
- LangChain 对话、精准寻源智能体

### 1.4 系统管理员

关注点：
- 登录认证状态
- 菜单可见性与 UI 风格策略
- 搜索工具和配额控制

常用页面：
- 搜索配置
- 菜单权限设置（可见性）
- UI 样式配置

## 2. 关键业务场景

1. 供应商主数据维护
- 列表筛选、批量处理、表单维护、关联链路查看。

2. 自动化采集与入库
- 根据任务配置执行爬取，查看日志和质量报告，导入业务库。

3. 知识沉淀与检索
- 文档上传/网页抓取/文本录入，随后使用检索或问答消费。

4. 智能辅助决策
- 在精准寻源中组织问题、选择工具、获得候选与证据。

5. 运维与配置治理
- 切换模型，管理外部服务，校验数据源，调整搜索策略。

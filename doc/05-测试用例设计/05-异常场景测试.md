# 异常场景测试

| 编号 | 场景 | 触发方式 | 预期系统行为 |
| --- | --- | --- | --- |
| EX-01 | 未登录访问受限 API | 不带 token 调 `/api/suppliers` | 返回未授权，前端跳登录 |
| EX-02 | 登录回调参数缺失 | 访问 callback 不带 code | 返回失败并可重新登录 |
| EX-03 | 数据库不可达 | 错误 DB 配置启动 | 健康检查显示异常，服务仍存活 |
| EX-04 | 模型配置错误 | provider key/url 错误后测试 | 返回明确错误信息 |
| EX-05 | 知识库文档解析失败 | 上传损坏文档 | 文档状态失败，错误信息可见 |
| EX-06 | 任务启动参数异常 | 创建非法任务后启动 | 返回参数错误，任务不进入运行态 |
| EX-07 | 任务执行中断 | 手动 stop/cancel 任务 | 状态变更为停止/取消，日志保留 |
| EX-08 | 任务导入失败 | 导入不匹配数据 | 返回失败原因，不污染既有数据 |
| EX-09 | Wiki 原始导入异常 | 无效内容执行 raw import | 返回失败记录，支持 resync |
| EX-10 | 数据源 SQL 非法 | 执行不受支持语句 | 返回错误，不执行破坏性操作 |
| EX-11 | MCP 服务不可用 | 安装或切换不存在服务 | 返回失败原因并保持原配置 |
| EX-12 | 搜索服务超时 | 执行 `/api/search-settings/test` 时超时 | 返回超时错误并可重试 |
| EX-13 | 流式对话中断 | 关闭网络或取消请求 | 服务正确结束流，前端提示中断 |
| EX-14 | 空数据页面 | 无匹配数据时打开列表 | 显示 Empty，不出现脚本错误 |
| EX-15 | 重复删除 | 连续删除同一记录 | 第二次返回合理提示，不崩溃 |

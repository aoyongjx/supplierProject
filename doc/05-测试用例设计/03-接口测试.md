# 接口测试

| 编号 | 类型 | 接口 | 断言点 |
| --- | --- | --- | --- |
| API-01 | 健康检查 | `GET /api/health` | 返回 code=200，含 DB 健康状态 |
| API-02 | 登录地址 | `GET /api/auth/login-url` | 返回可用跳转 URL |
| API-03 | 当前用户 | `GET /api/auth/me` | 登录态下返回用户信息 |
| API-04 | 资产分页 | `GET /api/inventories?page=1&pageSize=10` | 返回分页结构、数据数组 |
| API-05 | 任务详情 | `GET /api/crawl-tasks/:id` | 返回任务状态字段 |
| API-06 | 任务日志 | `GET /api/crawl-tasks/:id/logs` | 日志列表可读 |
| API-07 | 质量报告 | `GET /api/crawl-tasks/:id/quality-report` | 返回质量指标字段 |
| API-08 | 知识库检索 | `POST /api/knowledge-bases/:id/search` | 返回检索结果数组 |
| API-09 | Wiki 图谱 | `GET /api/llm-wiki/graph` | 返回 nodes/edges |
| API-10 | 模型联通 | `POST /api/model-providers/:providerName/test` | 返回成功或明确失败原因 |
| API-11 | 数据源 schema | `GET /api/data-sources/:id/schema` | 返回 schema/table 层级 |
| API-12 | 数据源 SQL | `POST /api/data-sources/:id/query` | 返回 columns/rows/elapsedMs |
| API-13 | LangChain 对话 | `POST /api/langchain/chat` | 返回 assistant 内容 |
| API-14 | RAG 对话 | `POST /api/langchain/rag-chat` | 返回答案并包含检索上下文信息 |
| API-15 | 精准寻源普通 | `POST /api/agents/precise-sourcing/chat` | 返回结构化响应 |
| API-16 | 精准寻源流式 | `POST /api/agents/precise-sourcing/chat-stream` | 流事件持续输出并正确结束 |
| API-17 | 菜单配置读写 | `GET/PUT /api/menu-visibility-settings` | 配置可保存并回读一致 |
| API-18 | UI样式读写 | `GET/PUT /api/ui-style-settings` | 配置可保存并回读一致 |
| API-19 | MCP管理 | `/api/mcp-services*` | 搜索、安装、切换、删除可用 |
| API-20 | Skill管理 | `/api/skills*` | 列表、安装、卸载可用 |

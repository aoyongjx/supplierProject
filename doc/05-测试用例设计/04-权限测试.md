# 权限测试

| 编号 | 场景 | 前置条件 | 步骤 | 预期结果 |
| --- | --- | --- | --- | --- |
| AUTH-01 | 未登录访问受限页面 | 清理本地 token | 直接访问业务路由 | 跳转登录页 |
| AUTH-02 | 未登录访问受限 API | 无 token | 调用 `/api/knowledge-bases` | 返回未授权 |
| AUTH-03 | 登录态访问权限 | 已登录 | 访问供应商、任务、配置接口 | 请求成功 |
| AUTH-04 | 登出后权限回收 | 已登录 | 执行登出并再次访问 API | 返回未授权并需重新登录 |
| AUTH-05 | 回调异常处理 | 构造无效 code | 访问 auth callback | 返回错误提示并不进入业务页 |

# 知识库管理

## 菜单与路由

- 路由：`/capability-center/knowledge-base`

## 功能说明

知识库与文档管理、预览与检索。

## 关键接口

- GET/POST/PUT /api/knowledge-bases*
- POST /api/knowledge-bases/:id/documents/file|web|text
- POST /api/knowledge-bases/:id/search

## 业务规则

- 文档状态应可追踪（处理中/成功/失败）。

## 验收标准

1. 页面可正常加载，无白屏。
2. 关键操作返回成功或明确错误提示。
3. 数据变更可在页面刷新后保持一致。

# 数据源管理

## 菜单与路由

- 路由：`/capability-center/data-source-management`

## 功能说明

数据源连接、结构浏览、SQL执行。

## 关键接口

- GET/POST/PUT/DELETE /api/data-sources*
- POST /api/data-sources/test
- GET /api/data-sources/:id/schema|table-preview
- POST /api/data-sources/:id/query

## 业务规则

- SQL 执行仅允许受控查询语句。

## 验收标准

1. 页面可正常加载，无白屏。
2. 关键操作返回成功或明确错误提示。
3. 数据变更可在页面刷新后保持一致。

# 模型管理

## 菜单与路由

- 路由：`/capability-center/model-management`

## 功能说明

模型提供商配置与模型拉取。

## 关键接口

- GET/POST/PUT/PATCH/DELETE /api/model-providers*
- POST /api/model-providers/:providerName/test|fetch-models

## 业务规则

- provider 名称重命名后引用保持一致。

## 验收标准

1. 页面可正常加载，无白屏。
2. 关键操作返回成功或明确错误提示。
3. 数据变更可在页面刷新后保持一致。

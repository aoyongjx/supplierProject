# 技术架构设计

## 目录目的

给出当前系统的可落地实现：总体架构、模块划分、数据库、接口、安全与运行方式。

## 核心架构结论

- 前端：React + Vite + Ant Design，按页面模块化拆分。
- 后端：Express 单体服务，按业务域组织接口。
- 数据：PostgreSQL/SeaboxSQL，服务启动自动初始化核心表。
- AI：模型提供商配置 + LangChain + 向量检索 + 精准寻源 Agent。

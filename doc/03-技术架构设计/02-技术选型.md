# 技术选型

## 1. 平台与运行时

- Node.js `^24.0.0`
- npm + Vite 构建
- Express 5 提供 API

## 2. 前端选型

- React 19、React Router
- Ant Design 6（中后台组件体系）
- 可视化组件：Recharts、Plotly、Force Graph、ProjectStorm Diagrams

选型价值：
- 快速构建复杂管理后台
- 适配多列表、多表单、多图谱页面

## 3. 后端选型

- Express + CORS + dotenv
- zod（参数结构化校验）
- 多业务域在同一服务内通过路由分组管理

## 4. 数据与检索选型

- PostgreSQL / SeaboxSQL
- `pg` 驱动
- 支持知识向量相关表与检索流程

## 5. AI / LangChain 技术栈

- `langchain`、`@langchain/core`、`@langchain/community`
- `@langchain/openai`（OpenAI 兼容模型接入）
- `@langchain/langgraph`（多步骤流程编排能力）
- `langsmith`（链路观测/追踪能力基础）

## 6. 多模态/文档处理能力

- `mammoth`（docx）
- `pdf-parse`（pdf）
- `exceljs`（xlsx）
- `jszip`（压缩包处理）
- `unstructured-client`（结构化提取辅助）

## 7. 自动化与外部访问

- `playwright`（网页渲染与自动化能力）
- MCP 服务管理（安装、启停、配置）

## 8. 选型原则

- 优先现成可靠组件，避免重复造轮子。
- 兼顾业务开发效率与 AI 扩展能力。
- 通过配置化降低后续新增模型/工具的改造成本。

# 系统模块设计

## 1. 前端模块（按路由）

### 1.1 基础与分析

- `/` 首页
- `/inventories*` 资产盘点
- `/stocks` 股票 K 线
- `/analytics` 分析看板

### 1.2 采集与业务主数据

- `/crawl-management`
- `/supply-chain*`、`/suppliers*`、`/supplier-profiles*`
- `/gas-supply-chain*`、`/gas-suppliers*`、`/gas-supplier-profiles*`
- `/gas-oems`、`/gas-industry-map`、`/users`（画像工作台）

### 1.3 智能体与能力中心

- `/agents/precise-sourcing`（实现）
- 其余 3 个 agent 路由为占位页
- `/capability-center/*`：MCP、Skill、知识库、向量检索、模型管理、搜索配置、数据源管理、LLM-Wiki

### 1.4 LangChain 与会话

- `/langchain-chatchat/multi-chat`
- `/langchain-chatchat/rag-chat`
- `/langchain-chatchat/knowledge-base`
- `/langchain-chatchat/mcp-management`
- `/sessions*`

## 2. 后端模块（按 API 前缀）

1. 鉴权模块：`/api/auth/*`
2. 配置模块：`/api/model-providers*`、`/api/search-settings*`、`/api/ui-style-settings`、`/api/menu-visibility-settings`
3. 插件模块：`/api/mcp-services*`、`/api/skills*`
4. 主数据模块：`/api/suppliers*`、`/api/gas-suppliers*`、`/api/gas-oems*`、`/api/supply-chain*`、`/api/gas-supply-chain*`
5. 任务模块：`/api/crawl-tasks*`、`/api/crawl-execute`、`/api/supplier-crawl-tasks*`
6. 知识模块：`/api/knowledge-bases*`、`/api/llm-wiki*`
7. AI 模块：`/api/langchain*`、`/api/agents/precise-sourcing/*`
8. 工具模块：`/api/data-sources*`、`/api/stocks*`

## 3. 模块协作关系

- 主数据模块为任务和知识模块提供实体主键与业务上下文。
- 任务模块为主数据模块补充外部采集结果。
- 知识模块与 Wiki 模块为 AI 模块提供检索语料。
- 配置模块影响 AI 模块的模型选择、工具可用性、前端导航可见性。

## 4. LangChain 模块特点

- 支持普通对话与 RAG 对话的分路调用。
- 支持会话状态持久化（session-state 接口）。
- 支持精准寻源普通与流式输出双通道。
- 支持工具清单动态获取（`/api/langchain/tools`）。

# 接口设计

## 1. 统一规范

- 协议：HTTP + JSON
- 前缀：`/api`
- 认证：默认 `authMiddleware`
- 响应结构：`{ code, message, data }`

## 2. 接口分组与功能清单

### 2.1 鉴权与会话

- `GET /api/auth/login-url`
- `GET /api/auth/callback`
- `GET /api/auth/me`
- `GET /api/auth/logout-url`
- `GET /api/sessions/recent`
- `GET /api/sessions`
- `POST /api/sessions`
- `GET /api/sessions/:id`
- `PATCH /api/sessions/:id`
- `DELETE /api/sessions/:id`
- `POST /api/sessions/:id/message`

### 2.2 资产盘点

- `GET /api/inventories`
- `GET /api/inventories/:id`
- `POST /api/inventories`
- `PUT /api/inventories/:id`
- `DELETE /api/inventories/:id`
- `DELETE /api/inventories/batch`

### 2.3 采集任务

- `GET /api/crawl-tasks`
- `POST /api/crawl-tasks`
- `GET /api/crawl-tasks/:id`
- `POST /api/crawl-tasks/:id/start`
- `POST /api/crawl-tasks/:id/stop`
- `POST /api/crawl-tasks/:id/retry`
- `GET /api/crawl-tasks/:id/logs`
- `GET /api/crawl-tasks/:id/quality-report`
- `POST /api/crawl-execute`
- `GET /api/crawl-exports/:fileName`

### 2.4 供应链与供应商

- GYS：`/api/supply-chain/*`、`/api/suppliers*`、`/api/supplier-profiles*`
- GAS：`/api/gas-supply-chain/*`、`/api/gas-suppliers*`、`/api/gas-oems*`

### 2.5 知识与 Wiki

- `GET/POST/PUT /api/knowledge-bases*`
- 文档子路由：`/documents/file|web|text|...`
- `POST /api/knowledge-bases/:id/search`
- `POST /api/llm-wiki/raw-import`
- `GET/POST/DELETE /api/llm-wiki/entries*`
- `GET /api/llm-wiki/graph` + `POST /api/llm-wiki/graph/sync`
- `GET/PUT /api/llm-wiki/settings`
- `GET /api/llm-wiki/sync/*` + `POST /api/llm-wiki/sync/:sourceType`

### 2.6 AI 与 LangChain

- `POST /api/langchain/chat`
- `POST /api/langchain/rag-chat`
- `GET /api/langchain/models`
- `GET /api/langchain/tools`
- `GET/PUT/DELETE /api/langchain/session-state`
- `POST /api/agents/precise-sourcing/chat`
- `POST /api/agents/precise-sourcing/chat-stream`

### 2.7 系统配置与平台能力

- 模型：`/api/model-providers*`
- 数据源：`/api/data-sources*`
- 搜索：`/api/search-settings*`
- UI/菜单：`/api/ui-style-settings`、`/api/menu-visibility-settings`
- MCP/Skill：`/api/mcp-services*`、`/api/skills*`

## 3. 错误与边界约定

- 参数错误：4xx
- 未授权：401/403（由鉴权中间件控制）
- 外部依赖异常：5xx + message
- 流式接口异常：保持连接结束前输出错误事件或可读错误消息
